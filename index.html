Getting Started

Untuk memulai Silakan masukkan suhu dalam Celsius atau Fahrenheit, kemudian klik tombol "Konversi" untuk melakukan konversi. Anda juga dapat menukar posisi suhu dengan menggunakan tombol "Reverse" dan mengosongkan semua input dengan tombol "Reset".

Perhatian

Kondisi berikut menyebabkan muncul pesan error jika:

  • Formulir input kosong
  • Formulir input terdapat huruf
  • Formulir input tidak valid
